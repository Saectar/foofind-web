Si te gustan los retos, el Software Libre y los vídeos de gatitos, únete a nosotros: gente joven de espíritu, con ganas de aprender, equivocarse y enfrentarse a problemas técnicos poco frecuentes. No tenemos buffet libre como en Google, pero bajamos al bar dos veces al día.

El trabajo es presencial y a jornada completa, en nuestra oficina de Madrid.

Ofrecemos incorporación a un proyecto de larga duración y flexibilidad de horarios.

Programadores Python

Buscamos programadores con experiencia en desarrollo con Python e interés en alguna de estas áreas:

  • Extracción de información de contenidos multimedia de Internet.
  • Análisis, extracción y manipulación de información semántica de la web.
  • Mejora de la experiencia del usuario, los resultados de las búsquedas, la comunidad de usuarios y el rendimiento del sitio web foofind.
  • Desarrollo de aplicaciones de escritorio multiplataforma.

Requisitos:

  • Python
  • Entornos Linux

También valoraremos conocimientos y experiencia en:

  • JavaScript, AJAX, jQuery y JSON
  • HTML y CSS
  • Acceso a APIs de terceros
  • Flask u otros frameworks para web
  • Sphinx o similar
  • MongoDB
  • wxWidgets
